> `Observer`（观察者）模式是一种设计模式，一个对象（称为subject）维护一系列依赖于它的对象，当状态发生任何变更时自动通知给它们。 

#### JavaScript实现：
```js
class Subject{
  constructor () {
    // 订阅主题的观察者
    this.subjects = {}
  }
  
  // 订阅
  subscribe (subject, observer) {
    if (!this.subjects[subject]) {
      this.subjects[subject] = []
    }
    this.subjects[subject].push(observe)
  }
  // 取消订阅
  unSubscribe (subject, observer) {
    const index = this.subjects[subject].indexOf(observer)
    this.subjects[subject].splice(index, 1)
  }
  // 发布通知
  notify (subject) {
    this.subjects[subject].forEach(observer => {
      observer.onMessageHandle()
    })
  }
}

class Observer {
  constructor (subject) {
    this.subject = subject
  }
  onMessageHandle () {
    console.log(`receive message from ${this.subject}`)
    this.subject = 'done'
  }
}

const subjects = new Subject()
const observer = new Observer('click')

subjects.subscribe('click', observer)
subjects.notify('click')
```
